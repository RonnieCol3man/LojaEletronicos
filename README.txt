Como executar o projeto:
- Abrir o netbeans
- Abrir o arquivo bdLojaEletronicos.sql
- Criar um banco de dados na aba serviços no netbeans com o nome "apd3"
- Copiar o conteudo do arquivo .sql e rodar o código
- Abrir na aba projetos os projetos web e aplicação java
- Abrir o projeto web e o pacote páginas web
- Abrir o index.jsp e clicar em executar
- O arquivo deve carregar uma página web no navegador
